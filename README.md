# 🚗 Autodiagnóstico de Vehículos

Una plataforma inteligente diseñada para facilitar el diagnóstico, reparación y seguimiento de incidencias mecánicas en vehículos.

---

## 📝 Descripción del Proyecto

Nuestra aplicación permite a los usuarios gestionar el mantenimiento y reparación de sus vehículos de forma integral. Al reportar un problema indicando el **modelo del coche**, el sistema proporciona:

*   **Diagnóstico asistido**: Identificación del posible problema mecánico.
*   **Gestión de repuestos**: Listado de piezas necesarias para la reparación o sustitución.
*   **Estimación económica**: Precio asociado a la intervención necesaria.

Además, si el usuario decide contratar el servicio, puede realizar un **seguimiento en tiempo real** de la reparación, conociendo en todo momento el estado del vehículo y posibles incidencias o retrasos.
